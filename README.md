This tool allows for precise hardware usage monitoring on Apple Silicon MacBooks. It is an open-source solution.

### Requirements
This is an app for Macs with Apple Silicon. It will not work on Macs with Intel processors. Do you have an older Mac? Consider using other open-source solutions.

Otherwise, if you have a MacBook with any M-series chip, you will only need to download [Homebrew](https://brew.sh/).

### Installation
**Brew**
```bash
brew trust bacteriafield/monitor
brew install monitor
```

**Curl**
```bash
curl -s https://raw.githubusercontent.com/bacteriafield/monitor/main/install.sh | bash
```
